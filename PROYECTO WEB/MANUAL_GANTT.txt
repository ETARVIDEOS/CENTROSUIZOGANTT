  MANUAL DE OPERACIÓN — CARTA GANTT INTERACTIVA
  Plan Digital Fase 1 | Centro Suizo Joyas
  URL: centrosuizogantt.netlify.app
════════════════════════════════════════════════════════

Al abrir la página siempre se cargan los últimos datos
guardados en la nube. No importa el navegador ni el
dispositivo — todos ven lo mismo.

────────────────────────────────────────────────────────
1. VER LA GANTT (modo normal)
────────────────────────────────────────────────────────
  → Las fechas de cada tarea se ven debajo del nombre
  → Los diamantes en la línea de HITOS son los hitos clave
  → Al hacer clic en un diamante baja automáticamente
    a la tarjeta del hito correspondiente
  → En este modo NO se puede cambiar nada (solo lectura)

────────────────────────────────────────────────────────
2. EDITAR FECHAS
────────────────────────────────────────────────────────
  → Marcar la casilla "✏️ Modo edición" (arriba a la derecha
    dentro del card de la Gantt)
  → Las fechas se convierten en campos editables
  → Hacer clic en la fecha → aparece el calendario del
    navegador → seleccionar la nueva fecha
  → La barra de la tarea se actualiza automáticamente
  → Para terminar de editar: desmarcar "Modo edición"

  ATAJOS DE TECLADO:
    Ctrl + Z  →  Deshacer (hasta 50 pasos)
    Ctrl + Y  →  Rehacer

────────────────────────────────────────────────────────
3. GUARDAR CAMBIOS EN LA NUBE
────────────────────────────────────────────────────────
  → Botón "💾 Guardar" (header, arriba a la derecha)
  → La PRIMERA VEZ pedirá el Token GitHub (ver sección 8)
    — ingresarlo y quedará guardado en el navegador
  → Luego pedirá la contraseña:

        centrosuizo2026

  → En ~1 minuto todos los usuarios ven los cambios
    al recargar la página

  IMPORTANTE: Sin hacer clic en Guardar, los cambios
  se pierden al cerrar o recargar la página.

  NOTA: El token se guarda en el navegador (localStorage).
  Solo se pide una vez por navegador/dispositivo. Si se
  borra el caché o se usa otro navegador, lo pedirá de nuevo.

────────────────────────────────────────────────────────
4. DESHACER / REHACER
────────────────────────────────────────────────────────
  → Botón "↩ Deshacer" — revierte el último cambio de fecha
  → Botón "↪ Rehacer" — vuelve a aplicar lo que se deshizo
  → Historial de hasta 50 pasos por sesión
  → También con Ctrl+Z y Ctrl+Y

────────────────────────────────────────────────────────
5. COMENTARIOS EN HITOS
────────────────────────────────────────────────────────
  → Activar "✏️ Modo edición"
  → Ir a la sección "Hitos Clave" (debajo de la Gantt)
  → En cada tarjeta aparece la sección de comentarios
  → Escribir el comentario y hacer clic en "Agregar"
  → Los comentarios se pueden editar (✏️) o eliminar (🗑)

────────────────────────────────────────────────────────
6. REINICIAR
────────────────────────────────────────────────────────
  → Botón "⟳ Reiniciar" (header)
  → Descarga el último estado guardado desde la nube
  → Descarta cualquier cambio no guardado de la sesión
  → Si no hay conexión, vuelve a las fechas por defecto

────────────────────────────────────────────────────────
7. IMPRIMIR / EXPORTAR A PDF
────────────────────────────────────────────────────────
  → Botón "🖨️ PDF" (header)
  → Se abre el diálogo de impresión del navegador
  → Seleccionar "Guardar como PDF" como destino
  → Orientación recomendada: Horizontal (Landscape)
  → Escala recomendada: 70-80% para que entre en una página

────────────────────────────────────────────────────────
8. CÓMO OBTENER UN TOKEN GITHUB (si lo pide de nuevo)
────────────────────────────────────────────────────────
  Necesitas estar logueado en github.com con la cuenta
  ETARVIDEOS.

  PASOS:
  1. Ir a: github.com/settings/tokens/new
  2. Note (nombre): gantt-centrosuizo
  3. Expiration: No expiration (o 1 year)
  4. Scopes: marcar solo ✅ repo (el primero de la lista)
  5. Clic en "Generate token" (botón verde al final)
  6. COPIAR el token que aparece (ghp_...)
     ⚠️ Solo se muestra una vez — copiarlo inmediatamente
  7. Ir a centrosuizogantt.netlify.app
  8. Clic en 💾 Guardar → pegar el token cuando lo pida
  9. Luego ingresar la contraseña: centrosuizo2026

  CUÁNDO necesitarás un token nuevo:
  - Si borras el caché del navegador
  - Si usas un navegador o dispositivo diferente
  - Si el token expira (si elegiste fecha de vencimiento)

RESUMEN RÁPIDO
════════════════════════════════════════════════════════
