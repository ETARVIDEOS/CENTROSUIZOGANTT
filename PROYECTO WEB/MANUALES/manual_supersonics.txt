1. ¿QUÉ ES SUPERSONICS ONE?
════════════════════════════════════════════════════════════

SuperSonics ONE es un CRM All-in-One que reemplaza más de 10
herramientas separadas. Según la plataforma misma, incluye:

  CRM y ventas:
  → Pipeline visual, programación automatizada de citas
  → WhatsApp API para cerrar ventas más rápido
  → Conversaciones: WhatsApp, Instagram, Facebook y Email centralizados

  Marketing digital:
  → Constructor de sitios web y embudos (funnels)
  → Email Marketing
  → Planificador de redes sociales
  → Administrador de anuncios (Facebook, Google, LinkedIn)

  Automatización:
  → Flujos de trabajo (workflows) por trigger
  → Automatizaciones por canal: Facebook, Instagram, WhatsApp

  IA:
  → Agentes de IA propios de la plataforma

4. INCRUSTAR FORMULARIO EN LANDING EXTERNA — CONFIRMADO ✅
════════════════════════════════════════════════════════════

SuperSonics SÍ permite embeber formularios en páginas externas
(incluyendo Cloudflare Pages y cualquier HTML estático).

CÓMO OBTENER EL CÓDIGO DE EMBED:
─────────────────────────────────
1. Ir a Sitios → Formularios
2. Abrir el formulario (o crear uno nuevo)
3. Clic en botón "Integrar" (arriba a la derecha del editor)
4. En el modal "Incrustar o compartir formulario":
   - Pestaña izquierda: "Incrustar código" (Embed Code)
   - Tipo de diseño: seleccionar "Inline"
   - Activador: "Always show"
   - Desactivación: "Never deactivate"
5. Copiar el código generado

TIPOS DE EMBED DISPONIBLES:
  ● Sticky sidebar    → barra lateral que sigue al usuario
  ● Polite slide-in   → aparece deslizándose desde el borde
  ● Popup             → ventana emergente con trigger
  ● Inline            → dentro del flujo de la página ← RECOMENDADO para landings

CÓDIGO DE EMBED REAL (Form 1 de Jordan):
──────────────────────────────────────────
<!-- SuperSonics Form Embed -->
<div style="height:740px; width:100%">
  <iframe
    src="https://api.supersonics.one/widget/form/Olr0hxVXrWqvv0C88Ykd"
    style="width:100%;height:100%;border:none;border-radius:8px"
    id="inline-Olr0hxVXrWqvv0C88Ykd"
    data-layout="{'id':'INLINE'}"
    data-trigger-type="alwaysShow"
    data-trigger-value=""
    data-activation-type="alwaysActivated"
    data-activation-value=""
    data-deactivation-type="neverDeactivate"
    data-deactivation-value=""
    data-form-name="Form 1"
    data-height="740"
    data-layout-iframe-id="inline-Olr0hxVXrWqvv0C88Ykd"
    data-form-id="Olr0hxVXrWqvv0C88Ykd"
    title="Form 1">
  </iframe>
</div>
<script src="https://api.supersonics.one/js/form_embed.js"></script>
<!-- Fin SuperSonics Form -->

URL DIRECTA del formulario (para compartir por WhatsApp/redes):
  https://api.supersonics.one/widget/form/Olr0hxVXrWqvv0C88Ykd

8. CONEXIÓN SUPERSONICS ↔ META ADS
════════════════════════════════════════════════════════════

Hay 2 formas de conectar SuperSonics con Meta Ads:

OPCIÓN A — Vía Administrador de Anuncios de SuperSonics (MEJOR)
──────────────────────────────────────────────────────────────
  → Activar el Ad Manager integrado (ver sección 6)
  → Conectar la cuenta de Facebook Ads de Jordan
  → Crear las campañas DESDE SuperSonics
  → Los leads caen automáticamente en el CRM
  → Sin Zapier, sin configuraciones adicionales
  → Ventaja: lead → CRM en tiempo real

OPCIÓN B — Vía Landing Page + Formulario Embed (PLAN ACTUAL)
──────────────────────────────────────────────────────────────
  → Anuncio en Meta Ads apunta a Landing en Cloudflare Pages
  → La landing tiene el formulario embed de SuperSonics
  → Cuando alguien llena el formulario → entra al CRM de Jordan
  → El Meta Pixel en la landing trackea el evento "Lead"
  → Ventaja: landing page propia con diseño personalizado

OPCIÓN C — Vía Zapier (para formularios nativos de Meta)
──────────────────────────────────────────────────────────────
  → Meta Business Suite ofrece Zapier como integración CRM
  → Zapier conecta "Meta Lead Form" → SuperSonics
  → Requiere plan Zapier + configuración adicional
  → Solo necesario si se usan formularios nativos de Meta (Lead Ads)

RECOMENDACIÓN PARA FASE 1:
  Usar OPCIÓN B (landing + form embed) para las campañas de
  Argollas y Anillos. Es la ruta más directa y ya está
  confirmada como funcional con el embed code real de SuperSonics.

  Paralelamente: ACTIVAR el Administrador de Anuncios (Opción A)
  para gestionar las campañas Meta desde SuperSonics directamente.
